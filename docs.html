IdeaMarkdown 文档

IdeaMarkdown 是一款桌面 Markdown 编辑器：编辑当前区块时显示源码，其余内容以排版后的形式展示，兼顾写作、笔记与轻量技术文档编辑。

本手册涵盖安装、编辑器功能、主题与导出，以及通过 MCP 与 CLI 让 AI 助手直接操作当前文档的完整用法。

安装

前往 下载页 获取 Windows 安装程序（.exe）。当前版本 v0.1.4，默认安装到 C:\Users\<用户名>\AppData\Local\IdeaMarkdown\。

安装包会同时附带 AI 控制所需的 ideamarkdown-mcp.exe 与 ideamarkdown-cli.exe，与主程序位于同一目录，无需单独下载。

安装完成后，双击任意 .md / .markdown 文件即可用 IdeaMarkdown 打开（已注册文件关联）。

编辑与区块

文档由一系列区块组成：标题、段落、列表、任务清单、引用、表格、代码块、公式与 Mermaid 图表。当光标进入某个区块时显示其 Markdown 源码，离开后立即渲染为排版态。

  • 拖动重排：左键拖动区块首部的把手可整体重排区块顺序。
  • 区块菜单：右键区块可复制或删除整个区块。
  • 任务清单：使用 - [ ] / - [x] 创建可勾选的待办项。
  • 引用：使用 > 开头创建引用块。

表格

表格支持丰富的鼠标交互：

  • 编辑单元格：双击进入编辑。
  • 多选：按住 Ctrl（macOS 为 Command）点击多个单元格。
  • 合并 / 拆分：右键菜单对选中的单元格执行合并或拆分，以及增删行列。
  • 拖动重排：通过行列专用把手拖动，重新排列行或列。
已合并单元格可以和相邻单元格继续合并；若仅选到已合并区域的一部分，操作会被拒绝以保护表格结构。

代码块

代码块提供语言提示、展示态语法高亮并保留缩进：

  • 使用围栏语法 ```lang 指定语言，例如 ```go、```ts、```python。
  • 展示态下按语言进行语法高亮，编辑态回到纯源码。
  • 代码过长时，按住 Shift + 鼠标滚轮可水平浏览。

斜杠命令

在空区块输入 / 唤起命令菜单，快速插入常用结构：一级至三级标题、加粗、斜体、删除线、行内代码、链接、无序 / 有序 / 任务列表、引用、代码块与分割线。菜单支持中英文关键字过滤。

公式与图表

内建 KaTeX 数学公式与 Mermaid 图表渲染。行内与块级公式在展示态自动排版，Mermaid 代码块渲染为流程图、时序图等图形，适合技术文档。

主题与字号

提供三种主题：跟随系统、亮色、暗黑；正文字号可在 12–18px 之间调整。偏好通过本地存储持久化，桌面端与 Web 预览通用。

导出 HTML

可将当前文档导出为样式内联的独立 HTML 文件，无需外部资源即可在任意浏览器中打开，便于分享与归档。

AI 控制（MCP + CLI）

IdeaMarkdown 可以由支持 MCP 的 AI 助手或命令行脚本控制当前打开的文档。

所有编辑都先进入窗口中的内存文档模型，可立即在界面中看到，并可按 Ctrl + Z 撤销；只有调用 save_document（CLI 中为 save）才会写入磁盘。

工作原理与安全

应用每次启动时会：

  1. 在随机端口启动仅绑定到 127.0.0.1 的 HTTP 控制服务。
  2. 生成新的随机会话令牌。
  3. 将发现文件写到 %APPDATA%\com.ideamarkdown.app\control.json：
{ "port": 51234, "token": "<uuid>" }

客户端读取该文件，然后向 http://127.0.0.1:<port>/control 发送请求，并带上 Authorization: Bearer <token>。服务只接受回环连接；令牌在每次启动时轮换，旧令牌随应用重启失效。

不要把发现文件中的令牌共享给不受信任的程序。
应用重启恢复：ideamarkdown-mcp.exe 和 ideamarkdown-cli.exe 会在第一次请求失败后自动重新读取 control.json 并重试一次，因此通常能在 IdeaMarkdown 重启后自动恢复。

MCP 服务器

MCP 服务器使用 stdio 传输，不需要参数；使用前应先启动 IdeaMarkdown。ideamarkdown-mcp.exe 随安装包附带，默认位于安装目录 C:\Users\<用户名>\AppData\Local\IdeaMarkdown\。

Cursor 配置

在 Cursor 的 mcp.json 中添加以下配置，并将路径替换为实际绝对路径：

{
  "mcpServers": {
    "ideamarkdown": {
      "command": "C:\\Users\\<用户名>\\AppData\\Local\\IdeaMarkdown\\ideamarkdown-mcp.exe"
    }
  }
}

Claude Desktop 配置

在 claude_desktop_config.json 中添加：

{
  "mcpServers": {
    "ideamarkdown": {
      "command": "C:\\Users\\<用户名>\\AppData\\Local\\IdeaMarkdown\\ideamarkdown-mcp.exe"
    }
  }
}

MCP 工具

工具参数作用
get_document 无 返回当前文档的 Markdown、区块数量和活动区块索引。
insert_block markdown（必填）、position（可选）、after_block_index（可选） 插入一个或多个 Markdown 区块。
insert_code_block lang（必填）、code（必填）、position、after_block_index 插入带语言标识的代码块。
replace_document markdown（必填） 用给定 Markdown 替换当前文档全部内容。
save_document 无 将当前文档保存到原路径；未保存的新文档会打开保存对话框。

参数说明

  • position：可取 atEnd、atCursor 或 afterBlock，默认 atEnd。
  • after_block_index：当 position=afterBlock 时使用的从零开始的区块索引；省略时为 0。

CLI 命令

CLI 位于安装目录 C:\Users\<用户名>\AppData\Local\IdeaMarkdown\ideamarkdown-cli.exe，随安装包附带，运行命令前应先启动 IdeaMarkdown。

命令作用
ideamarkdown-cli get读取当前文档
ideamarkdown-cli insert "<markdown>" [--position atEnd|atCursor|afterBlock] [--index N]插入 Markdown 区块
ideamarkdown-cli insert-code <lang> "<code>" [--position ...] [--index N]插入带语言标识的代码块
ideamarkdown-cli replace "<markdown>"替换整个文档
ideamarkdown-cli save将当前文档写入磁盘

--position 默认为 atEnd。--index N 仅在 --position afterBlock 时生效，N 是从零开始的区块索引；省略时为 0。

端到端示例：插入快速排序代码块

使用 AI 助手

  1. 启动 IdeaMarkdown 并打开一个 Markdown 文档。
  2. 按上文示例配置 MCP 服务器，然后重启或刷新 AI 客户端，使其加载 ideamarkdown。
  3. 对助手输入：“在打开的文档里添加一个 Go 语言代码块，写上快速排序算法”。
  4. 确认助手调用 insert_code_block，其中 lang 为 go、code 为快速排序实现，position 可设为 atEnd。
  5. 代码块应立即出现在窗口中。按 Ctrl + Z 可撤销；需要落盘时再让助手调用 save_document。

等价 CLI 命令

在 PowerShell 中保持 IdeaMarkdown 运行，并执行：

ideamarkdown-cli insert-code go `
  "func quicksort(a []int) []int { if len(a) < 2 { return a }; ... }" `
  --position atEnd

运行后检查窗口中是否出现 Go 代码块，并用 Ctrl + Z 验证撤销。这个实时窗口往返需要在可交互桌面会话中手动验证。

常见问题

为什么窗口只有部分区域能拖动？

应用采用无边框窗口，只有导航栏的空白背景区可以拖动窗口；菜单、文件名和窗口控制按钮保持可点击。

表格多选为什么用 Ctrl 而不是 Shift？

表格多选使用 Ctrl（macOS 为 Command），而不是 Shift。Shift + 滚轮用于代码块的水平浏览。

AI 客户端长时间连不上怎么办？

客户端会在首次失败后自动重读 control.json 并重试一次。若仍无法连接，请确认 IdeaMarkdown 正在运行，然后手动重启 MCP 服务器。
