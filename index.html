AI 控制 · MCP + CLI

让 AI 直接操作你打开的文档

IdeaMarkdown 可以由支持 MCP 的 AI 助手或命令行脚本控制当前打开的文档。所有编辑先进入内存文档模型，可立即在界面看到，并能按 Ctrl + Z 撤销；只有显式保存才会写入磁盘。

  • MCP 工具：get_document、insert_block、insert_code_block、replace_document、save_document。
  • 命令行 CLI：get / insert / insert-code / replace / save，脚本化你的文档流程。
  • 本地且安全：控制服务仅绑定 127.0.0.1，随机端口 + 每次启动轮换的会话令牌。
阅读 AI 控制指南 → 
// 对 AI 助手说：在文档里添加一个 Go 快速排序代码块
// 助手调用 MCP 工具 insert_code_block
{
  "tool": "insert_code_block",
  "arguments": {
    "lang": "go",
    "code": "func quicksort(a []int) []int { ... }",
    "position": "atEnd"
  }
}
# 保持 IdeaMarkdown 运行，然后在 PowerShell 执行
ideamarkdown-cli insert-code go "func quicksort(a []int) []int { ... }" --position atEnd

# 读取当前文档 / 落盘保存
ideamarkdown-cli get
ideamarkdown-cli save
